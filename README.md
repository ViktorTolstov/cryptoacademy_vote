# Demo task for CRYPTO ACADEMY

# # Install dependencies
```shell
npm i
```

# # Deploy contract script
```shell
npx hardhat run scripts/deploy.js --network rinkeby
```

# # Run tests
```shell
npx hardhat test
```

# # Standart hardhat shell:
```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help
```
